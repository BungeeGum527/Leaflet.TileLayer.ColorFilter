# Leaflet.TileLayerColorize
A simple and lightweight Leaflet plugin to apply CSS filters on map tiles.

# Basic Usage
  var colorSettings = {
    blur: '0px',
    brightness: '100%',
    contrast: '100%',
    grayscale: '0%',
    hue: '0deg',
    opacity: '100%',
    invert: '0%',
    saturate: '100%',
    sepia: '0%'
  }
